# SpaceX Launch Data Analysis

This project uses the SpaceX public API to fetch launch data and perform basic analysis using Python.

## Project Overview

- Fetch SpaceX launch, rocket, and launchpad data from the API
- Clean and transform the data
- Merge rocket and launchpad details for better readability
- Analyze success rates of launches
- Export cleaned data to CSV
